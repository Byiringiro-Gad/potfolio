# Monochrome Professional Portfolio

A clean, minimalist, and professional single-page portfolio built with HTML and CSS.

## Features
- **Strictly Monochrome Aesthetic**: Uses a high-contrast black, white, and gray color palette.
- **Responsive Design**: Fully functional on mobile, tablet, and desktop.
- **Modern Typography**: Integrated with Google Fonts (Inter).
- **Professional Icons**: Powered by Font Awesome.
- **Sections**: About, Projects, Skills, and Contact.

## How to Use
1. Clone or download this repository.
2. Open `index.html` in any modern web browser to view the portfolio.
3. Customize the content in `index.html` and styles in `styles.css` as needed.

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- Google Fonts
- Font Awesome
